# Install

1. Close Thunderbird
2. Download ZIP
3. Extract files
4. Run `install.bat`
5. Restart Thunderbird

--  
or copy `user.js` and full `/chrome` folder in your Thunderbird profile folder (usually `C:\Users\USERNAME\AppData\Roaming\Thunderbird\Profiles\xxxprofilenamexxx`)  
and  
Make sure the following config editor are set on `TRUE` :
 * toolkit.legacyUserProfileCustomizations.stylesheets
 * mail.identity.default.suppress_signature_separator
 * mailnews.attachments.display.start_expanded
 * mailnews.expand_threads

And in Settings/Composition
  * uncheck  "use paragraph format"
  * Font : choose "Calibri"


## Recommended Toolbar Setup

To get the intended appearance:

1. Right-click the Thunderbird toolbar
2. Select **Customize…**
3. Add the following buttons:
   - Write
   - Reply
   - Repy All
   - Forward
   - Delete
4. Click **Done**
